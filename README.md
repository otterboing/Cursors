 The `wait` and `progress` images are APNGs <img width="32" src="https://raw.githubusercontent.com/otterboing/Cursors/main/64px/progress.png" /> <img width="32" src="https://raw.githubusercontent.com/otterboing/Cursors/main/64px/wait.png" />

The `.mdp` files are opened in [FireAlpaca](https://firealpaca.com/)

I used [FSresizer](https://www.faststone.org/FSResizerDownload.htm) to resize the smaller versions.
